本プログラムは、vb.net で実装されたテストアプリです。
以下のように動作するアプリにしてください。
1. Form1.vb の [接続] ボタンを押すと、Form3.vb が起動する
2. Form3.vb では、ユーザーとパスワードの入力がある。そこに入力された値で「接続」ボタンを押すと、ユーザー、パスワードそれぞれの値がローカルの DB.ini に記録される。
3. 接続が成功したら、Form3.vb は閉じる
4. Form3.vb 起動時、すでに DB.ini ファイルが存在していれば、それぞれの値を画面上に表示する
5. Form3.vb のパスワードは入力内容は非表示（●で入力文字を埋める）
6. Form1.vb で「表示」ボタンを押されると、Form2.vbを表示する。
7. Form2.vb では、Form1.vb で選択された STEP_Scholl_Box の内容を WHERE 文に取り入れた SQL を発行し、その結果を Form2.vb のデータグリットに表示する。

いろいろ粗はありますが、いったんこの情報でコードを組めるかプランを立ててください。
